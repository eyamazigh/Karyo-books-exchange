# 📚 Karyō — Old Books Exchange Platform

> *"When we get, we give"*

Karyō is a front-end web platform that connects Tunisian students who have finished with their school books to those who need them — creating a bridge of solidarity between learners.

---

## 🌟 About the Project

The idea behind Karyō was born from a real problem: some students can't afford to buy books, while others have books they no longer need. Karyō acts as the bridge — making it easy to exchange old school books for free, helping students succeed together.

The website is designed primarily in Arabic (Tunisian dialect) and targets school students in Tunisia.

## 🚀 Getting Started

No installation needed. Simply open `acceuil.html` in your browser to launch the app locally.

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/karyo-books-exchange.git

# Open the project
cd karyo-books-exchange
open acceuil.html   # macOS
# or
start acceuil.html  # Windows
```
